### 遊戲說明

遊戲時間共 20 秒

點擊隨機出現的小女孩阿布

- 點到穿怪獸裝的小女孩加兩分
- 戴安全帽的和正常裝扮的小女孩加一分

點到怪獸的計分方式

- 毛怪、大眼仔每點一次，體積就縮小一次
- 點到藍道則會扣一秒，體積則變大
